# Network Traffic Analysis Lab (Wireshark)

## Objective
The objective of this lab is to capture and analyze network traffic using Wireshark to identify common protocols and understand how network communication works.

## Tools Used
- Wireshark
- Web Browser
- Terminal
- Windows / Linux

## Traffic Generated
During this lab the following protocols were analyzed:

- DNS
- TCP
- HTTP / HTTPS
- ICMP

## Lab Steps

1. Started packet capture using Wireshark.
2. Generated network traffic by visiting websites.
3. Used the ping command to generate ICMP traffic.
4. Applied filters to analyze different protocols.
5. Investigated packet details to understand communication patterns.

## Skills Practiced

- Network traffic analysis
- Packet inspection
- Protocol identification
- Basic network investigation
